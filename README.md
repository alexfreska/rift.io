# rift.io

The file goes from you to me.
The rest is pure magic.

## Getting Started
Install the module with: `sudo npm install -g rift.io`

```console
$ rift create image.png
Rift token: b97bab5e1449f9fc7b859583776503aa
Copied to Clipboard!
```
